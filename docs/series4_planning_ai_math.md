## 第四系列规划：数学如何驱动 AI（从直觉到方程）

### 1. 系列概述

- **系列名**：数学如何驱动 AI：从直觉到方程（英文：Math Powers AI）
- **定位**：以“可视化直觉”讲清 AI 核心数学（线代、概率、优化、信息论），替代枯燥推导
- **目标受众**：数学/编程爱好者、学生、数据/产品岗位，泛科普人群
- **平台**：抖音/视频号 3-4 分钟精华；B 站/YouTube 6-8 分钟扩展版
- **价值主张**：降低“黑箱”感，以可验证的动画与现实应用建立理解与信任

### 2. 爆款策略与风格

- **黄金 3 秒**：以“认知冲突/真实痛点”开场（例：为什么推荐总懂你？）
- **强可视化**：向量场/热力图/能量地形/矩阵分解/收敛曲线等可复用组件
- **可验证**：参数扫描、迭代收敛、随机模拟，给观众“看得见的证据”
- **现实应用回扣**：搜索/推荐、图像/音频、医疗/金融、自然语言

### 3. 视觉与组件库（复用要求）

- 色彩建议：主色 `#8B5CF6`（概率紫）、辅色 `#3B82F6`（数据蓝）、`#10B981`（成功绿）、`#F59E0B`（高亮黄）
- 通用组件（建议置于 `scenes/templates` 或 `scenes/ai_math/components`）
  - **MatrixDecomp**：矩阵分解/奇异值示意（SVD、低秩）
  - **Conv2DViewer**：卷积核扫描与特征可视化
  - **AttentionMap**：注意力热力图与加权示意
  - **EnergyLandscape**：能量地形/等高线+梯度下降路径
  - **ManifoldProjector**：PCA/t-SNE/UMAP 降维投影
  - **ConvergencePlot**：训练/优化收敛曲线

### 4. 选题清单（12 集，单集可独立观看）

1. 推荐系统的数学：相似度 → 矩阵分解（SVD/低秩）
2. 卷积看得见：滤波、边缘与特征（Conv2D）
3. 注意力为何“看重点”：加权与 Transformer 直觉
4. 词向量的语义空间：余弦相似与类比
5. 过拟合长什么样：偏差-方差的可视化折衷
6. 正则化在做什么：L1/L2 的几何意义与稀疏性
7. 梯度下降为何能收敛：能量地形与学习率
8. 损失函数的选择：MSE/交叉熵/鲁棒损失对比
9. PCA 主成分：高维到低维的最优投影
10. t-SNE/UMAP：邻近保持与簇结构的“幻觉”与真相
11. VAE 的魔法：重构与 KL 散度的直觉版
12. 扩散模型 Diffusion：从噪声到清晰的逆过程

### 5. 单集结构模板（3-4 分钟版）

- 0-5s：钩子（认知冲突/现实问题）
- 5-20s：可视化小实验/示例
- 20-120s：数学心脏（1-2 个关键方程+几何直觉）
- 120-180s：参数扫描/迭代收敛验证
- 180-210s：现实应用/误区澄清/边界条件
- 210-240s：总结金句 + 下集预告 + 互动引导

### 6. 制作流程与节奏（Windows / Manim）

1. 脚本与分镜（0.5-1 天）
   - 脚本：问题 → 直觉 → 数学 → 验证 → 应用 → 总结，100-150 行
   - 分镜：每 10-15s 一个“视觉高潮”
2. 动画实现（1-2 天）
   - 先搭“组件库”，再填具体镜头；复杂片段可预渲染
3. 渲染与发布（0.5 天）
   - 预览：`manim scene.py Scene -pql`
   - 发布：`manim scene.py Scene -pqh`（1080p）
   - 批量：`python scripts/batch_render.py`
4. 运营与复盘（持续）
   - 黄金 3 秒/封面 AB 测、评论问题收集、下一集预埋钩子
