# EP01 推荐系统的数学：相似度 → 矩阵分解

## 钩子（0-5s）

为什么推荐总懂你？答案在“相似度与低秩”。

## 示例（5-20s）

用户-电影评分表，空缺项需要预测。

## 数学心脏（20-120s）

- 低秩假设：偏好由少量潜在因子决定
- SVD 拆分：A ≈ U Σ V^T（只保留前 r 个奇异值）
- 直觉图景：行/列在潜在语义空间里更接近 → 更像

## 验证（120-180s）

- 留一法/交叉验证：误差下降意味着推荐更准

## 应用（180-210s）

电商/短视频/音乐/新闻：不同模态同一数学

## 总结与预告（210-240s）

今天：相似度 + 低秩分解；下集：卷积看得见。
