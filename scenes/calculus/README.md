# 数学函数之美系列 Function Aesthetics

目标：打造 1-3 分钟横屏短视频，通过参数可视化把抽象函数变成可收藏的数学艺术作品，主打年轻理工群体。

## 制作要点
- **节奏**：0-8s 钩子展示最终画面；8-45s 解析核心函数；45-105s 参数扫描；105-180s 现实应用或观众挑战。
- **视觉语言**：流体渐变背景 + 粒子尾迹；关键组件包含 `ParametricCurveBurst`、`FormulaMorpher`、`LevelSetRevealer`。
- **互动**：每集 CTA 邀请观众留言想看的函数或参数组合，并提供高清壁纸链接。
- **发布策略**：周更；抖音/视频号 60-120s 竖屏为主，B 站每月拼接合辑。
