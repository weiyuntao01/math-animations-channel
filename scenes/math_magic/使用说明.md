# 📱 数学之美系列 - 使用说明

## 🎯 系列简介

"数学之美"是专为短视频平台设计的数学科普系列，每个视频独立完整，用数学解读生活现象，发现数学与生活的美妙联系。

## 📂 文件说明

### 核心视频文件

- `math_magic_ep01.py` - **为什么总是拖延？** 数学公式告诉你答案
- `math_magic_ep02.py` - **为什么总买贵的？** 商家定价的数学陷阱
- `math_magic_ep03.py` - **为什么刷视频停不下来？** 上瘾的数学密码

### 配置文件

- `__init__.py` - 模块导入配置
- `render_all.bat` - Windows 批量渲染脚本
- `README.md` - 详细系列规划文档
- `优化总结.md` - 优化过程和效果总结

## ⚡ 快速使用

### 1. 渲染单个视频（预览）

```bash
# 拖延症公式
python -m manim scenes/math_magic/math_magic_ep01.py ProcrastinationFormula -pql

# 锚定效应
python -m manim scenes/math_magic/math_magic_ep02.py AnchoringEffect -pql

# 上瘾机制
python -m manim scenes/math_magic/math_magic_ep03.py AddictionMechanism -pql
```

### 2. 生产级渲染（发布用）

```bash
# 竖版高清60fps
python -m manim scenes/math_magic/math_magic_ep01.py ProcrastinationFormula -qh --resolution 1080,1920 --frame_rate 60
```

### 3. 批量渲染

```bash
# 运行批处理脚本
render_all.bat
```
