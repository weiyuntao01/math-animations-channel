# 数学之美系列 Math Beauty Series

《数学之美》是频道当前的主攻线：用真实案例与严谨模型，解释生活、财富、健康与社交中的复杂现象，并提供可复现的数据与模板。

## 系列定位
- **核心人群**：25-45 岁的理工背景观众、理性决策倡导者、关注家庭与职业规划的用户。
- **内容风格**：5-8 分钟深度短片，结构为「痛点 → 模型 → 验证 → 行动」。
- **视觉规范**：背景深灰；强调色使用 `#8B5CF6`、`#10B981`、`#F59E0B`；字体统一 `Microsoft YaHei`；尽量维持双栏布局（左图右文或上下切换）。
- **互动策略**：每集结尾提供小挑战或数据模板，引导评论区反馈；在公众号/社群发布行动清单。

## 制作规范
1. **脚本与分镜**：在 `docs/narration/math_beauty/epXX.md` 记录台本、镜头、数据来源与互动设计。
2. **组件复用**：公共动画组件存放于 `scenes/math_magic/components/`。修改组件需兼容既有场景，并在 README 中注明使用方法。
3. **数据与模板**：可下载资源统一放在 `media/toolkits/math_beauty/epXX/`，命名遵循 `epXX_topic_description.ext`。
4. **代码风格**：遵从项目整体约定（`black`、`flake8`、无中文 MathTex、半角标点、`Text.set_default(font="Microsoft YaHei")`）。
5. **审稿清单**：
   - 公式逐步推导、无偷步；
   - 动画时间轴明确，镜头切换不过度堆叠；
   - 仿真或实验代码可在 Manim 中复现；
   - 行动清单与互动问题与正文呼应。

## 渠道协作
- **Bilibili/YouTube**：上传完整版与字幕文件，链接到仓库数据。
- **抖音/视频号**：输出 60-90 秒短剪，聚焦“痛点 + 数学证据”。
- **公众号/社群**：整理为图文 + 表格模板，收集下一集选题反馈。
