# 数论奇观系列 Number Theory Wonders

定位：把整数规律可视化成霓虹灯与点阵艺术，让观众看到“最冷门的数学也能美得炸裂”。目标是 1-3 分钟横屏爆款、主打涨粉。

## 制作要点
- **结构**：0-8s 惊人画面/谜题；8-45s 揭示数论结构；45-105s 动态演示；105-180s 现实或趣味挑战。
- **视觉**：点阵、螺旋、模运算色块；常用组件 `LatticePulseGrid`、`NumberSpiralCollider`、`PrimeGlowField`。
- **互动**：评论“猜下一个点会亮在哪里”“提交你的整数序列我来可视化”；每月一次直播解析高赞评论。
- **发布**：与函数系列交替周更；抖音/视频号竖屏短片为主，B 站每月合集。
